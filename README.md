# KiGENsei Engine
### ✦ Your Visibility Origin
> A super-prompt system for generating cinematic landing pages from a company's brand, core info, and goals.
> No coding required. One session. One file. One LLM (AI) chatbot, in any browser.


---

## ✧ What is this?

The **KiGENsei Engine** is a structured AI prompt system that turns brand inputs into fully animated, production-quality landing page mockups — complete with scroll animations, interactive components, and a considered design system.

Paste the prompt into Claude or Google AI Studio, answer a few questions, get a single `.html` file back.

Open it in your browser, share it with a client, or drop it on Netlify for a live URL in 30 seconds.

**Built for:**
- ✧ Web designers building unsolicited client mockups
- ✧ Agencies creating lead magnets at speed
- ✧ Anyone who wants a premium web presence without a build process

---

## ✧ Quick Start

**Easiest path — no setup, no account beyond Claude:**

1. Open [claude.ai](https://claude.ai) — free account
2. Start a new chat
3. Open [`KiGENsei-V1.3.2`](./kigensei-engine-v1.3.2.md) or [`PROMPT-ONLY`](./PROMPT-ONLY.md) and copy everything between `## SYSTEM PROMPT START` and `## SYSTEM PROMPT END`
4. Paste into the chat
5. Type `build` — answer the questions — get your site

**For "best" output** — use [Google AI Studio](https://aistudio.google.com) with **Gemini 3.1 Pro Preview** and paste the system prompt into the System Instructions field.

Full instructions, platform guides, and mode explanations are inside the main file.
